### **Step 4: Connect to the New Database**

#### **Using pgAdmin (GUI)**

1. Expand **Databases** → Click **practice_db** → Start querying.

#### **Using psql (CLI)**

1. Switch to the new database:
   ```sql
   \c practice_db
   ```
   - Output:
     ```
     You are now connected to database "practice_db" as user "postgres".
     ```
